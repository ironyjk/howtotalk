## Faux Feelings / Pseudo-Feelings (가짜 느낌 — 실제로는 판단/해석)

These words LOOK like feelings but actually contain an evaluation of what someone ELSE is doing. Avoid using these in NVC:

| Faux Feeling | Hidden Judgment | Try Instead |
|---|---|---|
| abandoned (버림받은) | "You left me" | lonely, scared, hurt |
| abused (학대당한) | "You're abusing me" | angry, frightened, hurt |
| attacked (공격당한) | "You're attacking me" | scared, angry, tense |
| betrayed (배신당한) | "You betrayed me" | shocked, hurt, angry |
| blamed (비난받는) | "You're blaming me" | angry, scared, confused |
| bullied (괴롭힘당한) | "You're bullying me" | scared, angry, powerless |
| cheated (속은) | "You cheated me" | angry, hurt, disappointed |
| coerced (강요당한) | "You're forcing me" | resentful, scared, angry |
| dismissed (무시당한) | "You're dismissing me" | hurt, frustrated, lonely |
| ignored (무시당한) | "You're ignoring me" | lonely, hurt, sad |
| insulted (모욕당한) | "You insulted me" | angry, embarrassed, hurt |
| intimidated (위협당한) | "You're intimidating me" | scared, anxious |
| judged (판단받는) | "You're judging me" | tense, nervous, self-conscious |
| manipulated (조종당한) | "You're manipulating me" | angry, scared, powerless |
| misunderstood (오해받는) | "You don't understand me" | frustrated, discouraged |
| neglected (방치된) | "You're neglecting me" | lonely, sad, hurt |
| pressured (압박받는) | "You're pressuring me" | anxious, overwhelmed, resentful |
| provoked (자극받은) | "You provoked me" | angry, frustrated |
| rejected (거절당한) | "You rejected me" | hurt, scared, lonely |
| taken for granted (당연시 여겨지는) | "You take me for granted" | sad, hurt, frustrated |
| unappreciated (인정받지 못한) | "You don't appreciate me" | sad, discouraged, hurt |
| unsupported (지원받지 못한) | "You don't support me" | lonely, discouraged |
| used (이용당한) | "You're using me" | angry, sad, hurt |

## Quick Reference: The 4-Step Template

```
When I [see/hear/notice] ___ (observation),
I feel ___ (feeling),
because I need ___ (need).
Would you be willing to ___? (request)
```

## Key Distinctions

| Concept | This IS it | This is NOT it |
|---------|-----------|----------------|
| Observation | "You spoke for 20 minutes" | "You dominated the meeting" |
| Feeling | "I feel anxious" | "I feel you don't care" |
| Need | "I need predictability" | "I need you to be on time" |
| Request | "Would you text me by 5pm?" | "Stop being inconsiderate" |
