# howtotalk — AI를 위한 소통 프레임워크

13개의 소통·설득·대화 프레임워크, 하나의 진입점.
상황을 말하면 `/howtotalk`가 최적의 대화법을 자동 선택합니다.

## `/howtotalk` — 하나의 명령어로 13개 대화법 사용

```
/howtotalk 상사가 계속 내 공을 가로채는데 이걸 얘기해야 해
```

AI가 자동으로:
1. 상황 분석 (감정 수준, 권력 관계, 문화 맥락)
2. 최적의 프레임워크 1~3개 선택
3. 단계별 파이프라인 설계
4. 실제 사용할 대사와 표현 생성

## 명령어

| 명령어 | 설명 |
|--------|------|
| `/howtotalk` | 전체 분석 → 프레임워크 선택 → 스크립트 포함 전략 |
| `/howtotalk:select` | 빠른 프레임워크 추천만 (전체 전략 미생성) |
| `/howtotalk:practice` | 롤플레이 시뮬레이션 + 코칭 + 디브리핑 |

## 13개 프레임워크

### 설득 & 협상

| 프레임워크 | 창시자 | 용도 |
|-----------|--------|------|
| **NVC** | 마셜 로젠버그 | 비폭력 대화 — 비난 없이 욕구 표현 |
| **결정적 대화** | 패터슨, 그래니 등 | 감정이 격해지는 중요한 대화 |
| **차이를 만드는 협상** | 크리스 보스 (전 FBI) | 전술적 협상 — 연봉, 거래 |
| **Yes를 이끌어내는 협상** | 피셔 & 유리 (하버드) | 윈윈 협상, BATNA |
| **설득의 심리학** | 로버트 치알디니 | 설득의 7원칙 — 마케팅, 조작 감지 |

### 피드백 & 리더십

| 프레임워크 | 창시자 | 용도 |
|-----------|--------|------|
| **래디컬 캔더** | 킴 스콧 (전 구글/애플) | 아끼면서 직설적으로 피드백 |
| **어려운 대화** | 스톤, 패턴, 힌 (하버드) | 미뤄왔던 대화 시작하기 |
| **동기면담** | 밀러 & 롤닉 | 변화를 거부하는 사람 움직이기 |

### 발표 & 교육

| 프레임워크 | 창시자 | 용도 |
|-----------|--------|------|
| **스토리텔링** | 캠벨, 듀아르테, 픽사 | 프레젠테이션, 피치, 내러티브 |
| **소크라테스식 문답** | 소크라테스 | 가르치기, 코칭, 질문으로 인사이트 |

### 자기인식 & 경청

| 프레임워크 | 창시자 | 용도 |
|-----------|--------|------|
| **SCARF** | 데이빗 록 | 뇌과학 — 팀 역학, 위협/보상 이해 |
| **적극적 경청** | 칼 로저스 | 진짜 듣기 |
| **DESC** | 바우어 부부 | 자기주장 — 경계 설정 |

## 구조

```
/howtotalk (메타 라우터 — 사용자 직접 호출)
└── 13개 프레임워크 (내부용, /howtotalk이 자동 선택)
    ├── nvc, crucial-conversations, never-split
    ├── getting-to-yes, influence, radical-candor
    ├── difficult-conversations, motivational-interviewing
    ├── storytelling, socratic
    └── scarf, active-listening, desc
```

13개 프레임워크는 `user-invocable: false`로 설정되어 슬래시 메뉴에 나타나지 않습니다. `/howtotalk`가 내부적으로 Skill 도구를 통해 호출합니다. 상황만 설명하면 라우터가 알아서 처리합니다.

## 멀티 프레임워크 파이프라인

복잡한 상황은 여러 프레임워크를 순서대로 사용합니다:

| 상황 | 파이프라인 |
|------|-----------|
| 어려운 대화 | 어려운 대화 → NVC → 적극적 경청 |
| 딜 협상 | Yes 협상 → 차이를 만드는 협상 → 설득의 심리학 |
| 피드백 제공 | SCARF → 래디컬 캔더 → 적극적 경청 |
| 그룹 설득 | 설득의 심리학 → 스토리텔링 → 소크라테스 |
| 팀 갈등 해결 | 결정적 대화 → NVC → Yes 협상 |
| 변화 저항 동기 부여 | SCARF → 동기면담 → 소크라테스 |
| 우아하게 거절하기 | 적극적 경청 → DESC → NVC |

## 사용 예시

```
/howtotalk 팀원이 계속 마감을 어기는데 지적하면 방어적이야
→ SCARF (위협 진단) + 래디컬 캔더 (전달) + 동기면담

/howtotalk 연봉 협상에서 20% 인상을 받고 싶어
→ Yes 협상 (BATNA) + 차이를 만드는 협상 (전술) + 설득의 심리학 (마무리)

/howtotalk:practice 고객의 무리한 요청 거절하기
→ 인터랙티브 롤플레이 + 교환마다 코칭
```
